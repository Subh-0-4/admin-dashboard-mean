# Admin Dashboard with Analytics & Reporting

## Overview
This project is a web-based **Admin Dashboard** developed using the **MEAN stack**.  
It provides administrators with a clean and intuitive interface to view analytics, monitor key metrics, and analyze overall system performance.

The application emphasizes usability, responsive design, and clear data visualization, similar to real-world **CRM and analytics admin panels**.

---

## Tech Stack
- **Frontend:** Angular 17 (Standalone Components)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Charts & Visualization:** Chart.js
- **Styling:** Custom CSS (no heavy UI frameworks)

---

## Key Features
- Admin login workflow (mock authentication for demo purposes)
- Dashboard with analytics summary cards
- Interactive charts for user growth and sales trends
- Responsive UI for desktop and mobile devices
- Clean, modular, service-based architecture

npm install
ng serve

The frontend application will be available at:
http://localhost:4200

## Backend Setup
cd backend
npm install
node server.js

The backend API will run at:
http://localhost:5000
