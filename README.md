Admin Dashboard with Analytics & Reporting
Overview

This project is a web-based Admin Dashboard built using the MEAN stack.
It provides a clean and intuitive interface for administrators to view analytics, monitor key metrics, and understand system performance.

The application focuses on usability, responsive design, and clear data visualization, similar to real-world admin panels used in CRM and analytics platforms.
Tech Stack

Frontend: Angular 17 (Standalone Components)

Backend: Node.js, Express.js

Database: MongoDB

Charts & Visualization: Chart.js

Styling: Custom CSS (no heavy UI frameworks)

Key Features

Secure admin login (mock authentication for demo purposes)

Dashboard with analytics summary cards

Interactive charts for user growth and sales trends

Responsive UI for desktop and mobile

Clean service-based architecture (Angular services + REST APIs)
